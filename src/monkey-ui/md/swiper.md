# mk-footer

## 使用示例
```JavaScript
  // 默认   swiperList-图片列表
  <mk-swiper type='default' :swiperList='swiperList'></mk-swiper>

  // 走马灯   swiperList-图片列表
  <mk-swiper type='notable' :swiperList='swiperList'></mk-swiper>
```

| 属性           | 类型(默认值)     | 可选值           | 备注                |
| ------------- | ------------- | ----------       | ------------------  |
| type          | string("")    | default/notable  | 默认/走马灯           |
